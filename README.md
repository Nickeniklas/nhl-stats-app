# NHL Stats Web Application

**Node.js, API Integration, HTML/CSS**

## API
`https://api-web.nhle.com/v1/`

## Setup

1. Clone repository
   `git clone https://github.com/Nickeniklas/nhl-stats-app/`
   
3. Go to project root folder
   `cd nhl-stats-app`
   
5. Install packages
   `npm install`
   
7. Run server
   `node server.js`
   or in Dev mode `nodemon server.js`
